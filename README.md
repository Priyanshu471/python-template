# This is the starter template for a python project.

## It follows the below structure.
```
project_name/
│
├───src/
│ └───__init__.py
│ └───main.py
│
├───tests/
│ └───__init__.py
│ └───test_main.py
│
├───setup.py
├───requirements.txt
└───README.md
```
## We can add `server`, `static`, `data` and `docs` directories to this structure.
