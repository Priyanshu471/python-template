# This is the starter template for a python project.

## It follows the below structure.

project_name/
│
├───src/
│ └───**init**.py
│ └───main.py
│
├───tests/
│ └───**init**.py
│ └───test_main.py
│
├───setup.py
├───requirements.txt
└───README.md

## Additionally we can also add `server`,`static`,`data` and `model` directories to this structure.
